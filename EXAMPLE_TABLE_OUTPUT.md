## Notes on Table Formatting

The GT package will create professional, publication-ready tables with:
- Bold headers
- Center-aligned prevalence values
- Left-aligned stratification and category labels
- Consistent column widths
- Professional typography
- Clear footnotes with EAR values and data source

These tables are saved as PNG images in the `figures/` directory and can be directly inserted into reports, presentations, or publications.
