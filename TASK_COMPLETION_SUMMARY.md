### 2. Main Script: `src/13DHS_triangulation.R`
A complete, production-ready R script (289 lines) that:

#### Data Processing
- Loads EICV7 survey data (base apparent intake, household information, administrative boundaries)
- Sources EAR values from `src/05base_model_functions.R`
- Calculates binary inadequacy indicators for Vitamin A and Iron
- Creates survey design object with proper weights using `srvyr` package

#### Analysis
Generates survey-weighted prevalence estimates for:
- **National level**: Overall Rwanda estimate
- **Provincial level (ADM1)**: 5 provinces
  - City of Kigali
  - Eastern Province
  - Northern Province
  - Southern Province
  - Western Province
- **Residence**: Urban and Rural
- **Socioeconomic status**: Quintiles 1 (poorest) through 5 (wealthiest)

#### Table Generation
Creates two separate GT summary tables:

**Vitamin A Table**
- Title: "Risk of Inadequate Vitamin A Intake in Rwanda"
- EAR threshold: 490 μg RAE
- 13 rows showing all stratifications
- Saved as: `figures/vita_inadequacy_table.png`

**Iron Table**
- Title: "Risk of Inadequate Iron Intake in Rwanda"
- EAR threshold: 22.4 mg (low absorption)
- 13 rows showing all stratifications
- Saved as: `figures/fe_inadequacy_table.png`

#### Table Features
- Publication-ready formatting
- Bold headers and proper alignment
- Footnotes explaining EAR values and data source
- Consistent with existing `08overall_inadequacy.R` style
- Professional typography and layout
